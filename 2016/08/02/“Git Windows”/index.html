这就是正确答案，我们已经验证过了，下面详细描述一下解决方法：

  1. 在Windows中添加一个HOME环境变量，值为%USERPROFILE%，如下图：

  2. 在“开始》运行”中打开%Home%，新建一个名为“_netrc”的文件。

  3. 用记事本打开_netrc文件，输入Git服务器名、用户名、密码，并保存。示例如下：
    machine git.cnblogs.com
    login cnblogs_user
    password cnblogs_pwd
