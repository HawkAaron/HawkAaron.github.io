mysql 远程数据库导入到本地数据库

Posted on 2015-05-15   |   In MySQL 
mysqldump -h192.168.0.1 -uNAME -pPASS --opt --compress DBS --skip-lock-tables | mysql -h127.0.0.1 -uroot -proot DBD

解释：

参数 含义
192.168.0.1 远程服务器IP
NAME 远程数据库登录名
PASS 远程数据库登录密码
DBS 远程数据库名（即：复制的源）
127.0.0.1 本地数据库服务器IP
root 本地数据库登录名（一般情况下都是这个）
root 本地数据库登录密码（一般情况下都是这个）
DBD 本地数据库名（即：复制的目的）

SQL解释：

选项 含义
mysqldump mysql的一个专门用于拷贝操作的命令
–opt 操作的意思
–compress 压缩要传输的数据
–skip-lock 忽略锁住的表（加上这句能防止当表有外键时的报错）
-tables 某数据库所有表
-h 服务器名称
-u 用户名（后面无空格，直接加用户名）
-p 密码（后面无空格，直接加密码）

注意：

-h、-u、-p的后面没有空格，直接加IP、用户名和密码！！！
